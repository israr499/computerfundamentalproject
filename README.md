### Run Locally

To run **Fitlife** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/israr499/computerfundamentalproject.git
```

Windows:

```bash
git clone https://github.com/israr499/computerfundamentalproject.git
```
